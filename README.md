# ⚽ SoccerScore

### Descrição

O **SoccerScore** é um aplicativo Android desenvolvido em **Kotlin** utilizando **Jetpack Compose** e **Navigation Compose**.  
O projeto simula um placar de futebol, permitindo que o usuário informe os dados de uma partida, visualize um resumo e confirme o resultado final.

O aplicativo foi desenvolvido com foco no uso de componentes básicos de interface, navegação entre telas e passagem de parâmetros por rotas.

---

### Tecnologias Utilizadas

- Kotlin
- Android Studio
- Jetpack Compose
- Navigation Compose
- Material 3
